# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Be civil. Assume good faith. No harassment, no slurs, no trolling. Disagree on the work, not the person.

## Reporting

If you experience or witness behavior that breaks this code:

1. Open a [private security advisory](https://github.com/MoizIbnYousaf/marketing-cli/security/advisories/new). It is private to the maintainer and works as a confidential reporting channel for conduct issues too.
2. Or contact the maintainer at the address on the [GitHub profile](https://github.com/MoizIbnYousaf).

You will get a response within 7 days. The maintainer reviews each report and decides on action: warning, removal of contributions, or ban from the repo. The decision and reasoning go back to the reporter.

## Scope

This applies to all project spaces: issues, pull requests, discussions, code review threads, and any public communication that represents the project.
