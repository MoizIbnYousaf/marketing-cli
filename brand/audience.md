# Audience

## Primary ICP
Solo founders and agent builders shipping products with Claude Code / Cursor / Codex who need marketing that compounds instead of one-off ChatGPT threads.

## Jobs to be done
- "My agent forgets our brand every session"
- "I need consistent launch/SEO/social output without hiring a CMO"
- "I want skills my agent can actually run, not blog posts about marketing"

## Watering holes
- Twitter/X (AI builders, indie hackers)
- Hacker News (Show HN, agent tooling threads)
- GitHub (README-driven discovery)
- Discord/Slack communities around Claude Code / Cursor

## Messaging angles
- Compounding brand memory
- One install, agent-ready
- CLI for the agent, Studio for the human
