# Creative Kit

## Visual direction
- Dark, dense, terminal-adjacent — not purple SaaS gradients
- Accent: emerald/mint on near-black (matches README badges)
- Typography: monospace signals for CLI; clean sans for human Studio UI

## Motifs
- Terminal prompt + `/cmo`
- Skill count badges
- Brand folder glyph (`brand/`)

## Do not
- Generic robot mascots
- "AI sparkle" purple glows
