## How it works
1. `npm i -g marketing-cli`
2. `mktg init` — install skills/agents, scaffold `brand/`
3. Write the minimum memory (voice, positioning, audience)
4. Ask `/cmo` (or a single skill) to ship files under `marketing/`
5. Log learnings so tomorrow starts smarter

**CLI for the agent. Studio for the human.**

## What it is not
- Not a social scheduler SaaS (schedulers can be adapters)
- Not a one-paste prompt pack
- Not "AI that writes your LinkedIn" as the whole product
- Not a claim that we replace a full enterprise marketing org on day one

## Differentiation (falsifiable)
1. Skills work at L0 (zero brand context); brand memory sharpens them — never a gate
2. `/cmo` orchestrates marketing work; skills read/write files instead of calling each other
3. Upstream catalogs over HTTP keep license firewalls clean (no AGPL SDK surprise)
