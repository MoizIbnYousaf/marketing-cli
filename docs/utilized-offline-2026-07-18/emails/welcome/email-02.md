# Email 02 — Minimum brand memory

**Send:** Day 1  
**Subject:** 10 minutes in brand/ > another clever prompt  
**Preview:** voice, positioning, audience — ugly drafts welcome.

```
Subject: 10 minutes in brand/ > another clever prompt

Your agent can run marketing skills at zero context. That is intentional.

If you want output that sounds like you, spend ten minutes on three files:

1) brand/voice-profile.md
   - Personality in 3 bullets
   - Words you prefer / refuse
   - Two example lines you would ship

2) brand/positioning.md
   - One-liner
   - Anti-positioning (what you are not)
   - Three concrete differentiators (counts, commands, files)

3) brand/audience.md
   - ICP in one sentence
   - Top job-to-be-done in their words
   - Two watering holes

Skills work at zero context. Brand memory makes them sharper — never a gate.

Do not polish. Truth beats poetry. Commit the files so the next session starts warmer.

— mktg
```
