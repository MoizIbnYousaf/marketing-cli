# Follow-up Emails — Agent Marketing Setup Checklist

Sequence for leads who downloaded the checklist. Plain-text friendly. Voice: direct, proof-first.

**Timing:** Email 1 = immediate (delivery). Email 2 = Day 2. Email 3 = Day 5.

---

## Email 1 — Delivery (immediate)

**Subject:** Your 30-minute agent marketing checklist  
**Preview:** Install → brand memory → one skill. That's the loop.

```
Subject: Your 30-minute agent marketing checklist

You asked for the Agent Marketing Setup Checklist — here it is.

[Download / view checklist link]

The path is short on purpose:

1) npm i -g marketing-cli
2) mktg init
3) Fill voice + positioning + audience (ugly drafts are fine)
4) Run ONE skill to completion under marketing/
5) Log one learning in brand/learnings.md

Skills work at zero context. Brand memory makes them sharper — never a gate.

If you only have ten minutes today, do steps 1–3. Tomorrow's agent session will already be smarter.

Repo: https://github.com/MoizIbnYousaf/marketing-cli

— mktg
```

## Email 2 — Brand memory nudge (Day 2)

**Subject:** The part most people skip (and why output stays generic)  
**Preview:** Empty brand templates = generic marketing files.

```
Subject: The part most people skip (and why output stays generic)

Quick check: did you fill brand/voice-profile.md and brand/positioning.md yet?

If those are still templates, your agent will ship competent-but-generic drafts. That is not a model failure — it is missing memory.

Minimum pass (about 10 minutes):
- Personality + words you refuse
- One-liner + 3 anti-positioning lines
- ICP in one sentence + top job-to-be-done

Then run one skill only. Finish the files. Commit brand/ + marketing/ together.

Install reminder:
npm i -g marketing-cli && mktg init

Checklist again: [link]

— mktg
```
