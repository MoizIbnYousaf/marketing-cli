# Newsletter Strategy — mktg / Agent-Native Marketing

## Purpose
Compound trust with builders who install (or almost install) marketing-cli. Each issue should leave one concrete play they can run with an agent — not a link dump.

## Audience
Primary: solo founders + agent builders on Claude Code / Cursor / Codex.  
Secondary: small product teams experimenting with agent-run marketing ops.

## Positioning for the list
"Playbooks for agent-native marketing" — proof, installs, brand memory patterns. Avoid growth-hack theatrics.

## Cadence
| Phase | Frequency | Length |
|-------|-----------|--------|
| Launch (issues 1–4) | Biweekly | 600–900 words |
| Steady | Monthly | 700–1,000 words |
| Incident / release | As needed | ≤400 words |

## Content pillars (rotate)
1. **Install / first 30 minutes** — commands, doctor checks, first `/cmo` run  
2. **Brand memory compounding** — what to put in `brand/`, freshness, anti-patterns  
3. **Orchestration stories** — `/cmo` routes, parallel research agents  
4. **Tool routing** — Exa / AXI / catalogs without AGPL contamination narratives  
5. **Launch playbooks** — Show HN, directories, atomized social  

## Format (every issue)
1. One-line cold open (problem or proof)  
2. Single deep section (the play)  
3. Ship it (commands / checklist)  
4. What we're not claiming (anti-hype)  
5. CTA: install or star + reply prompt  

## Metrics that matter
- Install referrals from issue UTM (`?utm_source=newsletter`)  
- Reply rate / "I ran this" screenshots  
- GitHub stars attributed to issue week  
- Unsubscribe rate (keep <0.5% by staying useful)

## Channels
- Primary: email (tool TBD — Resend/Buttondown/etc.)  
- Mirror: truncated X thread + full archive on site/GitHub Discussions if needed  

## List growth (ethical)
- README + npm page soft CTA  
- End of Studio empty-state (optional, dismissible)  
- Lead magnet: Agent Marketing Setup Checklist (see dogfood assets)  
- No bought lists. No dark-pattern prechecks.

## Editorial voice
Direct, product-obsessed, builder-friendly. Prefer: ship, compound, playbook, agent-native, dry-run, brand memory, orchestrate.
