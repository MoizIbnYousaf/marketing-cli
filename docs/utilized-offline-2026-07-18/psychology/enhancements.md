# Psychology Enhancements — Landing Headline & CTA

Applied to mktg hero (README / future landing). Voice stays direct and proof-first — principles sharpen clarity; they don't add hype.

**Baseline (before):**  
"AI marketing tools for developers."  
CTA: "Get started"

---

## Principles applied

1. **Specificity effect** — concrete counts beat adjectives  
2. **Goal gradient / next-step clarity** — show the exact install command as the CTA  
3. **Identity congruence** — speak to "coding agent builders," not "marketers"  
4. **Progressive commitment** — L0 promise reduces perceived setup cost (related to effort justification / friction removal)

---

## Before → After

### Headline

| Before | After | Principle |
|--------|-------|-----------|
| AI marketing tools for developers | Your coding agent already ships product. Now give it a marketing playbook. | Identity + specificity of job |
| Unlock AI-powered growth | Marketing that compounds across sessions — brand memory included. | Specificity (avoid banned hype) |

**Recommended hero headline:**  
**Your coding agent already ships product. Now give it a marketing playbook.**

### Supporting line

| Before | After | Principle |
|--------|-------|-----------|
| Everything you need to market with AI | Install one CLI. 64 skills, 6 research agents, brand memory that compounds. | Specificity effect |

### Primary CTA

| Before | After | Principle |
|--------|-------|-----------|
| Get started | `npm i -g marketing-cli` | Goal gradient — the action *is* the install |
| Learn more | View on GitHub (secondary only) | Commitment: don't compete with primary |

### Microcopy under CTA

| Before | After | Principle |
|--------|-------|-----------|
| Free trial · No credit card | Skills work at zero context. Brand memory sharpens — never gates. | Progressive commitment / friction removal |

## What we intentionally did *not* use

| Pattern | Why skipped |
|---------|-------------|
| Fake scarcity ("only 50 licenses") | OSS; unethical + false |
| Social proof walls of logos we don't have | No fabricated customers |
| Curiosity gaps that hide the product | Agents need predictable CTAs |
| Fear-based "you'll fall behind" | Off-voice |
