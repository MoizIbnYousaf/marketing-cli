## Link hygiene
- Canonical: GitHub README or docs install page  
- UTM on directory links: `utm_source={directory}&utm_medium=listing`  
- Anchor text variety: product name, “marketing-cli”, “agent marketing playbook” — avoid exact-match spam  

## 30-day definition of done
- [ ] Show HN posted  
- [ ] ≥5 directories live or submitted  
- [ ] ≥2 awesome-list or listicle mentions  
- [ ] npm + GitHub hero match CRO after-copy
