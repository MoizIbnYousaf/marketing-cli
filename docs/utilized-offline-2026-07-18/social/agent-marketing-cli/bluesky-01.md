# Bluesky — 01

Coding agents are great at shipping product.

Marketing still evaporates when the session ends — unless brand lives in files the agent can re-read.

mktg is an agent-native marketing playbook CLI: 64 skills, 6 agents, persistent `brand/` memory, `/cmo` to orchestrate.

Skills run at zero context. Memory sharpens — never gates.

`npm i -g marketing-cli`

CLI for the agent. Studio for the human.
