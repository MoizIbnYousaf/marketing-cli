# LinkedIn — 01 · The session reset problem

Your coding agent ships product fine.

Marketing? Still a fresh chat every Monday — re-explaining voice, audience, and what you already tried.

That isn't a model problem. It's a memory problem.

**mktg** is an agent-native marketing playbook CLI:

- 64 versioned skills (manifest-backed, tested)
- 6 research agents your orchestrator can spawn in parallel
- Persistent `brand/` memory that compounds across sessions

Skills work at zero context. Brand memory makes them sharper — never a gate.

CLI for the agent. Studio for the human (`mktg studio` ships in the same tarball).

```bash
npm i -g marketing-cli
mktg init
```

If you build with Claude Code / Cursor and you're tired of re-briefing marketing every session — this is the install.
