# LinkedIn — 02 · Progressive enhancement > gated onboarding

Most "AI marketing" tools ask you to fill a 40-field brand questionnaire before they'll write a sentence.

Agents hate that. So do founders shipping alone.

mktg's rule is progressive enhancement:

1. **L0** — skills run with zero brand files
2. **L1–L4** — as `brand/` fills in (voice, positioning, audience, keywords…), every skill gets sharper
3. Brand memory **enhances**, never **gates**

`/cmo` orchestrates the foundation: research agents write `voice-profile.md`, `audience.md`, `competitors.md` into your project — real files, not chat residue.

Dry-run before mutations. JSON when piped. `--fields` when context windows matter.

That's agent DX you can test — not vibes.

Install: `npm i -g marketing-cli`

Docs live on GitHub under marketing-cli. Star if the playbook model clicks for your stack.
