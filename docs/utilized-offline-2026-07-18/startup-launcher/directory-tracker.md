**Long (≤1000 chars)**  
mktg is an agent-native marketing playbook for builders using Claude Code, Cursor, and similar coding agents. One install (`npm i -g marketing-cli`) provides 64 versioned marketing skills, 6 research/review agents, and a `brand/` memory layer that compounds across sessions. Skills work at zero context; brand files sharpen output without gating the first run. `/cmo` orchestrates marketing work; `/axi` routes agent-ergonomic tools. A local Studio dashboard launches via `mktg studio` from the same npm tarball. Built for agent DX: JSON when piped, `--dry-run` on mutations, and field filtering for context discipline. Not a social scheduler — schedulers can act as adapters. Not a paste-once prompt pack — skills are manifest-backed and testable.

CLI for the agent. Studio for the human.
