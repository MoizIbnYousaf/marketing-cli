# Agent Quality Rubric

Screenshot every 2-3 writes. After each screenshot, rate against these criteria. If any criterion fails, fix BEFORE moving to the next slide.

## 6-Point Checklist

### 1. Text Readability
Can every word be read at 50% zoom?
- Body text: minimum 48px (TikTok/Story) or 36px (carousel)
- Hero text: minimum 120px (TikTok/Story) or 80px (carousel)
- If text is too small, increase immediately — readability is non-negotiable

### 2. Brand Fidelity
Does this look like the brand?
- Colors match palette exactly (check hex values, not "close enough")
- Font family and weights match creative-kit.md
- Voice rules respected (no hype words, correct tone)
- Signature phrases used where appropriate

### 3. Archetype Match
Does this slide match its content spec type?
- `stat` → big number is the BIGGEST element on the slide
- `anchor_word` → one word dominates, everything else is secondary
- `cascade` → sequential lines with clear vertical rhythm
- `cta` → centered, ornamental, URL prominent
- `emotional_pivot` → generous whitespace, centered, breathing room

### 4. Safe Zones
Is content within platform safe zones?
- TikTok: center 70% (avoid top 150px, bottom 300px)
- Instagram carousel: 5% padding all edges
- Instagram story: avoid top 10% and bottom 20%

### 5. Visual Rhythm
Does this slide feel different from the previous one?
- Alternate backgrounds (navy / navy-deep)
- Vary scale (huge headline → small body → huge stat)
- Different slide types in sequence (stat → anchor → cascade, not stat → stat → stat)

### 6. Hierarchy
Is the most important element the most visually dominant?
- Hero element should be 3-5x larger than secondary elements
- One focal point per slide (not two competing elements)
- Information flows: hero → supporting → detail
- Gold accent draws the eye to the key moment

## Quick Decision Tree

```
Screenshot taken. What's wrong?

Text too small?           → Increase font size, re-screenshot
Wrong colors?             → Check creative-kit.md, fix hex values
Slide looks like previous → Change background shade, vary layout
Two elements competing    → Make one 3x larger, reduce the other
Content near edges        → Add padding, move to center zone
Doesn't match archetype   → Redesign to emphasize the right element
```

## Post-Build Final Review

After all slides are built, take one final full-artboard screenshot and check:
- [ ] Consistent footer across all slides
- [ ] Background alternation creates visual rhythm
- [ ] First slide has the strongest visual hook
- [ ] Last slide has clear CTA with URL
- [ ] No two adjacent slides have the same layout structure
- [ ] Brand palette used throughout (no off-brand colors crept in)
